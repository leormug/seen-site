Mac Instructions

  1. Download the .dmg file using the link above.
  2. Open the downloaded file and drag SEEN into your Applications folder.
  3. If macOS blocks it:
    • Open System Settings → Privacy & Security.
    • Find “SEEN was blocked…” and click Open Anyway.
  4. Open SEEN from your Applications folder. It runs offline and stores everything locally.

Windows Instructions

  1. Download the .exe installer using the link above.
  2. If Chrome says “Not commonly downloaded,” click Keep.
  3. If Windows SmartScreen appears:
    • Click More info, then Run anyway.
  4. Complete the installer. Open SEEN from your desktop or Start menu.

Phone & Tablet Instructions (iOS & Android)

  1. Open your browser and go to:
    https://yourdomain.com/app
  2. Add SEEN to your home screen:
    • iPhone/iPad: tap Share → Add to Home Screen.
    • Android: tap the menu (⋮) → Install App or Add to Home Screen.
  3. Open SEEN from your home screen. It behaves like an app and keeps data on your device.

Using SEEN

  • On first launch, create a password to protect your data on your own device.
  • Enter your symptoms, history, medications, tests, and reasons for your visit.
  • Use the print option to generate a clean one-page summary for your doctor.
  • All information is stored locally. Updates to SEEN will not erase your data.
